# Exercise 3 - Selectors Playground

Objective:
Master different CSS selector types and apply styles using IDs, Classes, Elements, Universal Selectors, and Grouping Selectors.

Topics Covered:
- Universal Selector (*)
- Element Selector
- ID Selector (#mainHeader)
- Class Selector (.eventCard)
- Grouping Selector (h3, p)

Files:
- index.html
- styles.css

Outcome:
This exercise demonstrates how different CSS selectors can be used to target and style specific HTML elements efficiently.
